Tessera

A comprehensive open-source voxel game developed over several years. Built from the ground up using Java and LWJGL (OpenGL), the project features a custom rendering pipeline and efficient data storage and memory handling.

Features include:
  • Systems: LAN Multiplayer, Day/Night cycles, and Random Tick Events.
  • Lighting: Real-time Torch and Sun light propagation logic.
  • Physics: Fire spreading mechanics and falling gravity-affected blocks (Sand/Gravel).
  • Agriculture: Dynamic crop growth and Amethyst crystal maturation systems.
  • Exploration: Procedural unique biomes, cave systems, and ore distribution.
  • Transportation: Functional Minecart rail systems and navigable boats.
  • Fauna: Diverse entity AI (Dogs, Cats, Foxes, Beavers) with biome-specific spawning.
  • Survival: Survival crafting, loot tables, recipes and furnace and storage blocks.

Cogni-Crawl

Cogni Crawl is a flashcard memorization game built in Godot. This game turns memorization into a monster-hunting adventure. Find your way through a procedurally generated maze, fight enemies, and escape the dungeon!

Feed the game any JSON flashcard set you want — With a data driven approach, you can create your own customized flashcard set, upload it to CogniCrawl, and play your facts!

The game comes pre-included with Multiplication, Addition, Subtraction and Musical notes.
